# E-Learning Platform

A web-based E-learning platform built using Python and Django.

## Features
- User Registration & Login
- Course Management
- Admin Panel
- Responsive Design
- Category-wise Courses

## Technologies Used
- Python
- Django
- HTML
- CSS
- Bootstrap
- SQLite

## How to Run the Project

1. Clone the repository
2. Install requirements
3. Run: python manage.py runserver
